# Garden Advice App

## Overview

This project is a simple JavaScript application that provides gardening advice based on a selected season and plant type.

The application demonstrates basic JavaScript concepts, including:

* Variables
* Functions
* Conditional statements (`if` / `else`)
* User input using `prompt()`
* Console output

## Features

* Provides gardening advice based on the season.
* Provides gardening advice based on the plant type.
* Accepts user input for season and plant type.
* Uses functions to improve readability and modularity.

## Technologies Used

* HTML5
* JavaScript

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/DyllanMeissenheimer23/garden-app.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Enter a season and plant type when prompted.

5. Open the browser console to view the generated gardening advice.

## Example Input

Season:

```text
summer
```

Plant Type:

```text
flower
```

## Example Output

```text
Water your plants regularly and provide some shade.
Use fertiliser to encourage blooms.
```
